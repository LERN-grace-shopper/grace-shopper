 ## LERN norms:
 ---
  Share the mic
  Don't interrupt - raise your hand**
  Share frustrations in a constructive matter
  Write a GitHub issue
  Ask for help
  Commit frequently
  Deploy early
  Choose a code style and stick to it
  Keep each other updated
  Disagreements many, arguments few
